Platform capabilities
CSV data upload
Drag-and-drop with PapaParse client-side preview, column type detection, and server validation up to 500,000 rows.
Data explorer
4-tab explorer: column overview, type profiling, null analysis (colour-coded bars), and interactive correlation heat map.
Visualisations
Histogram, box plot, scatter, CDF, and pair grid powered by Recharts with configurable bins and column selection.
Preprocessing wizard
4-step pipeline: imputation (mean/median/KNN/drop), scaling (standard/min-max/robust/log), feature selection, and summary review.
A/B model testing
Compare classification and regression models with radar chart, metric bars, training speed comparison, and automatic winner detection.
CI/CD pipeline
Path-based change detection, 80%+ coverage gate, BDD + unit tests, Trivy container scan, staging auto-deploy, production manual gate.
Social authentication
Google and Facebook OAuth — tokens verified server-side. Automatic account linking by email. No redirect URLs needed — SDK popup flow.
